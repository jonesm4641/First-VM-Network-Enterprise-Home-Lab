Enable SSH on Windows


1. Make sure OpenSSH Server is installed
Get-WindowsCapability -Online | Where-Object Name -like 'OpenSSH*'
Add-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0


2. Switch "sshd" service to automatic
Start-Service sshd
Set-Service -Name sshd -StartupType Automatic

3. Make sure SSH is allowed through Firewall (Make Rule if it doesn't exist, make sure Port 22 is listening)!
New-NetFirewallRule -DisplayName "OpenSSH Server" -Direction Inbound -Protocol TCP -Action Allow -LocalPort 22
